# 스프링 HATEOS 
 - REST API 를 더 쉽게 만들기 위해 만들어주는 프레임워크라고 생각하면 됨 
 - 링크 만드는 기능
    - 문자열 가지고 만들기
    - 컨트롤러와 메소드로 만들기 
 - 리소스 만드는 기능
    - 리소스: 데이터 + 링크
 - 링크 찾아주는 기능
    - Traverson
    - LinkDiscovers
 
 - 링크
